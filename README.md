# Machine Learning Projects

A collection of **beginner-friendly Machine Learning projects** designed to understand core ML concepts such as **data preprocessing, model training, prediction, and evaluation**.

Each project focuses on a different **type of ML algorithms**.

## Overview

This repository contains multiple **mini ML projects** built using **Python** and popular libraries like **scikit-learn**, **pandas**, and **numpy**.

These projects are ideal for:
- Students learning ML basics  
- Building a strong **GitHub portfolio for placements**  
- Practicing ML concepts and interview preparation  

##  List of Projects

| No. | Project Name | Type | Algorithm Used | Description |
|:--:|:-----------------------------|:----------------|:--------------------|:------------------------------------------------|
| 1 | Iris Flower Classification | Classification | Random Forest | Predicts species of iris flowers using petal and sepal features. |
| 2 | House Price Prediction | Regression | Linear Regression | Predicts housing prices using the California housing dataset. |
| 3 | Student Score Prediction | Simple Regression | Linear Regression | Predicts student marks based on number of study hours. |
| 4 | Weather Prediction | Classification | Decision Tree | Predicts weather (Sunny/Rainy) using temperature and humidity data. |
| 5 | Credit Card Fraud Detection | Classification | Logistic Regression | Detects fraudulent transactions using binary classification. |
| 6 | Wine Quality Prediction | Classification | Random Forest | Predicts wine quality based on chemical properties. |
| 7 | Diabetes Prediction | Classification | Logistic Regression | Predicts diabetes likelihood based on patient medical data. |
